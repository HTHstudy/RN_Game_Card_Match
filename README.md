# RN_Game_Card_Match

### React native 로 Card Match 게임을 구현했습니다.

### 데모 영상 링크

https://youtu.be/3H5pPNH135g

### 구현 사항

- [x] 오픈된 카드가 2장 이상일 때 다른 카드 flip 불가능
- [x] 오픈된 2장의 카드의 그림이 다를 경우 카드 close
- [x] 오픈된 2장의 카드의 그림이 같을 경우 카드 오픈된 상태로 유지

### Quick Start

```bash
npm run ios
npm run android
```

또는

```bash
yarn ios
yarn android
```
